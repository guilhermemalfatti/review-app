# Indica — frontend

React + Vite + TypeScript UI for condo trusted-provider recommendations (Cantegril).

## Run

```bash
npm install
npm run dev
```

Dev server proxies `/api` → `http://localhost:8080` (cookie sessions work same-origin).

## Build

```bash
npm run build
npm run preview
```

Copy is Portuguese (Brazil). API contract lives in the monorepo backend agent brief.
